Manage links in the menu of MemberPress account page.

== Description ==

This plugin allows you to manage the links available in the user account pages of MemberPress.

== Installation ==

Search for "MemberPress Menu" in the WordPress.org plugin directory and install it.

After activating the plugin, the configuration can be accessed via MemberPress / Links settings.

Note: the [MemberPress](https://memberpress.com/) plugin must be installed and activated.

= Manual Installation =

1. Upload the entire `/igotweb-wp-mp-links` directory to the `/wp-content/plugins/` directory.
2. Activate MemberPress Menu through the 'Plugins' menu in WordPress.
3. MemberPress Menu configuration can be accessed via MemberPress / Links settings
